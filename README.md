#Simple LDO Converter

#A compact +5V to +3.3V regulator module using the MIC5317-3.3YM5-TR LDO.

#Features
Input: +5V
Output: +3.3V
Low dropout voltage
Compact PCB design

#Files
Schematic – Circuit diagram
PCB Layout – Board design
Usage
Connect J1 to a +5V source and get +3.3V output at J2.
